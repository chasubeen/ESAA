# **1. datetime 오브젝트**
- datetime 라이브러리: 날짜와 시간을 처리하는 등의 다양한 기능을 제공하는 파이썬 라이브러리
- 날짜를 처리하는 ```date``` 오브젝트, 시간을 처리하는 ```time``` 오브젝트, 날짜와 시간을 모두 처리하는 ```datetime``` 오브젝트가 포함되어 있음
- 시간 계산이 가능

### **1-1. datetime 오브젝트 사용하기**
- datetime 라이브러리 불러오기
```Python
from datetime import datetime
```

- 현재 시간 출력
  - ```now()``` 밀리초까지 구해준다.
  - ```today()```
  - 시간을 **직접** 입력하여 인자로 전달 가능

### **1-2. to_datetime() 메서드**
- 문자열 -> datetime 오브젝트
- ```format``` 인자에 **시간 형식 지정자(%d, %m, %y)**와 **기호(/,-)**를 적절히 조합하여 전달하면 해당 형식에 맞게 정리된 datetime 오브젝트를 얻을 수 있음
- 시간 형식 지정자

- ```strftime()``` 메서드와 시간 형식 지정자를 조합하여 시계열 데이터를 원하는 단위까지 잘라낼 수 있음

### **1-3. read_csv() 메서드**
- datetime 오브젝트로 변환하려는 열을 **지정**하여 데이터 집합을 불러오는 방식
- ```parse_dates``` 파라미터 활용

### **1-4. 날짜 정보 추출하기**
- datetime 오브젝트에는 년, 월, 일과 같은 날짜 정보를 따로 저장하고 있는 속성이 이미 존재함
- ```dt.year```: 연도
- ```dt.month```: 달
- ```dt.day```: 일

### **1-5. dt 접근자**
- datetime 속성이나 메서드를 활용하여 시계열 데이터를 처리할 수 있음
- ex>
```Python
ebola['year'] = ebola['date_dt'].dt.year
```
- ```df.quarter```: datetime 속성에서 분기 파악 가능

### **2-1. datetime 오브젝트와 인덱스**
- ```DateTimeIndex```: datetime 오브젝트를 인덱스로 지정
  - 원하는 시간의 데이터를 바로 추출할 수 있음

### **2-2. 시간 간격과 인덱스**
- ```Timedeltaindex```: 시간 간격을 인덱스로 지정하여 데이터를 추출

### **2-3. 시간 범위와 인덱스**
